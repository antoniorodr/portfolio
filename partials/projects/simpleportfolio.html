Flex Portfolio

A responsive and minimalist portfolio website built using Flexbox layout. Designed to showcase projects and skills in a clean and accessible manner.

Key achievements and learnings:

  • Emphasized accessibility and semantic HTML to enhance user experience and SEO performance.

  • Gained hands-on experience in creating a real-world, production-ready static website.
