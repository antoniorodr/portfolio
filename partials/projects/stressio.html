Stressio

A digital platform for measuring and understanding stress levels in the workplace. Employees can report stress anonymously while managers receive aggregated insights to make informed decisions and improve the work environment.

My role: Solo developer responsible for the full stack of the platform. Built the frontend using Tailwind CSS and DaisyUI, and implemented the backend with Python and FastAPI. Managed database migrations with Alembic and added internationalization support using Babel.

Key achievements and learnings:

  • Developed a fully functional full-stack application independently, from database to frontend.
  • Designed interactive and responsive dashboards for real-time stress reporting.
  • Ensured data privacy and anonymity, fully compliant with GDPR.
  • Implemented database migrations and multilingual support, improving maintainability and accessibility.
  • Gained extensive experience in project planning, full-stack development, and delivering a polished, production-ready solution.
